# RRT Playground
Implementation of Rapidly Exploring Random Trees (RRT) path planner and its some of its popular variants (RRT*, Informed RRT*) in modern C++.

## Installation
SFML and Eigen.
```
$ sudo apt-get install libsfml-dev libeigen3-dev
```
